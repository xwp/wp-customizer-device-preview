== Description ==

This is a temporary polyfill plugin for a WordPress.com feature while waiting for it to be [added to VIP Quickstart](https://github.com/Automattic/vip-quickstart/issues/405).

The JS and CSS assets in this plugin were copied from WordPress.com's CDN and de-obfuscated.
